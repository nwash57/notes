##### Oh My Posh
- Install Oh My Posh
	`winget install JanDeDobbeleer.OhMyPosh -s winget`
- Install Font
	`oh-my-posh font install`

##### Posh-Git
- Install
	`PowerShellGet\Install-Module posh-git -Scope CurrentUser -Force`
